# 扫雷概率计算器 - 前沿区优化
[立即使用](https://ycryv.github.io/MineSweeperSolver-Frontier/)
- **程序为AI生成**
- 基于前沿格子优化的扫雷概率计算器，通过识别并只枚举前沿区，大幅提升计算效率。

### ⚠️注意事项
- **非前沿区概率仅供参考，计算出的概率与真实概率可能有偏差**
- 棋盘无大小限制，前沿格过多可能导致卡死
- 填入的**数字0**并不表示周围没有雷，而是**标记非雷格**

### 前沿区优化算法
- 自动识别**前沿格**（与数字相邻的未知格）和**非前沿格**
- 只枚举前沿区的组合，非前沿区通过组合数计算

### 概率分析
- **整体概率**：雷数未知下所有可能布局下的平均概率
- **按总雷数分布**：查看不同雷数下的条件概率
- **概率可视化**：不同概率区间用不同颜色标记
  - 绿色(0%) / 红色(100%) 高亮显示
  - 概率百分比显示在格子右上角

### 操作
- **键盘支持**：WASD/方向键移动选中格子
- **高亮反馈**：选中的数字/格子有蓝色边框提示

### 填数方式
- 选中格子 → 点击数字按钮
- 点击数字按钮（高亮）→ 连续点击格子填数 → 再次点击同一数字取消

## 技术原理

1. **前沿识别**：找出所有与数字相邻的未知格
2. **约束构建**：根据数字建立约束方程
3. **深度优先搜索**：枚举前沿区的组合
4. **组合数计算**：处理非前沿区的雷分布
5. **概率统计**：汇总所有可能布局的概率
